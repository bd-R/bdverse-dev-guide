Title:

An Infrastructural Toolkit for Biodiversity Data Quality in R

Description:

A sustainable and agile infrastructure that enhances the value of biodiversity data by allowing users to conveniently employ R, for data exploration, quality assessment, data cleaning, and standardization. The bdverse supports users with and without programming capabilities. It includes six unique packages in a hierarchal structure â€” representing different functionality levels.
